Floods are not Fun

What?

This is a testing app playing about with flooding.

How Much?

At the moment the flood function works, but isn't tied to the view.

The view can display a given array assuming it's two-dimentional and is properly formatted.

Paste in a new array and submit, and the view will update.

How Much More?

In the next sprint, I'll be getting the view to call and display info from the flood function.
